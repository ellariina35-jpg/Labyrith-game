# 🎮 Labyrinth Game

A tiny dungeon escape game built with **Godot Engine** as a learning experiment.

## 📖 About

This is a simple dungeon escape game created for educational purposes. Navigate through the dungeon and find your way out!

## 🎯 Objective

**Escape the dungeon!** Tap the door to proceed to the next area and ultimately escape.

## 🕹️ How to Play

- **Controls**: Mouse only
- **Objective**: Navigate through the dungeon
- **Action**: Click/tap the door to proceed to the next room
- **Goal**: Escape the dungeon!

## 🚀 Running the Game

Simply run the executable:
```
Labyrinth Espace.exe
```

**System Requirements:**
- Windows (x86_64)
- Mouse

## 🛠️ Built With

- [Godot Engine](https://godotengine.org/) - Game engine
- GDScript - Scripting language

## 📁 Project Structure

```
├── Main.gd / Main.tscn       - Main game scene and logic
├── Player.tscn               - Player character
├── Door.tscn                 - Door/transition mechanics
├── player_character.gd       - Player behavior
├── door.gd                   - Door behavior
├── project.godot             - Godot project configuration
└── Labyrinth Espace.exe      - Compiled game executable
```
